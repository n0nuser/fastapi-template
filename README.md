## Getting Started

Create a `.env` file in the `api` directory of the project. Fill the example below with your own values.

```shell
backlog=2048
debug=False
host="0.0.0.0"
log_level="trace"
port=8080
reload=True
timeout_keep_alive=5
workers=4
```

Install the Python dependencies:

```shell
pip3 install -r requirements.txt
```

Launch the server:

```shell
cd api
python3 api.py
```
